# vlad.dev portfolio

Single-page portfolio built as a self-contained static site for quick preview and easy editing.

## Files

- `index.html` - structure and content
- `styles.css` - visual system, layout, and responsive behavior
- `script.js` - smooth state handling for header, section highlighting, reveal effects, and tilt interactions

## Open locally

Open `index.html` directly in a browser.
